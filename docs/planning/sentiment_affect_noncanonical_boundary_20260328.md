# Sentiment and Affect Non-Canonical Boundary (2026-03-28)

## Purpose
Record the refreshed boundary from `Sentiment Analysis in Docs` so the repo
does not drift into treating sentiment, affect, or psychometric-style
inference as canonical truth-bearing legal semantics.

This is a planning note. It sharpens existing transcript/freeform and
candidate-only doctrine rather than introducing a new runtime lane.

## Main Decision
Keep sentiment and affect below the truth gate unless a future lane defines a
fresh, explicit promotion contract.

Current safe reading:

- sentiment is not legal truth
- affect is not legal truth
- speaker/utterance/discourse anchoring is mandatory
- any emitted affect-like structure is candidate, bridge, or overlay material
  only

## Allowed current use
- source-local, anchored discourse observations
- candidate-only `felt_state`-style relations
- additive bridge inputs for later comparison / review pressure
- clearly non-canonical overlays in transcript or review surfaces

## Forbidden current drift
- no psychometric inference as canonical semantics
- no legal-practice dashboard posture based on inferred mood/state
- no negotiation-weighting or public-consultation authority transfer from
  sentiment analytics
- no claim that sentiment scores define obligation, liability, or legal
  meaning

## Strong proving case
If the repo wants a stronger future lane, the proving case should be bounded
and structurally explicit:

- affidavit / response pairs
- claim / response pairs
- speaker-anchored discourse units with explicit source spans

Even there, the first useful outputs are:

- discourse-level candidate observations
- contradiction / corroboration pressure
- abstention when speaker/claim structure is not secure

## Followthrough
- use this note alongside
  `docs/planning/transcript_semantic_phase_v1_20260308.md`
- keep `felt_state` and similar cues candidate-only
- keep any later dashboard or bridge use explicitly additive and
  non-authoritative

## Non-Goals
- not a commitment to building sentiment analytics now
- not a permission slip for mood-driven canonical promotion
- not a claim that affect-style overlays are useless; only that they remain
  subordinate to anchors, receipts, and explicit authority boundaries
